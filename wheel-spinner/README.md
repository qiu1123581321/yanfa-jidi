# 🎡 转盘抽奖器

一个精美的 Chrome 扩展插件，支持自定义奖品、加权概率、流畅动画。

## 功能特点

- 🎯 **加权抽奖** — 每个奖品可设置权重，越重概率越高
- 🎨 **精美界面** — 深色背景 + 金色点缀，专业配色
- ⚡ **流畅动画** — 五次缓动曲线，自然减速
- ✏️ **实时编辑** — 在插件内直接增删改奖品
- 💾 **本地存储** — 奖品配置保存在本地，换电脑后需重新配置

## 安装方式

1. 下载本仓库（或克隆到本地）
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本仓库中的 `wheel-spinner` 文件夹

## 使用方法

1. 点击 Chrome 工具栏中的插件图标，弹出抽奖界面
2. 点击「开始抽奖」转动转盘
3. 点击「编辑奖品」自定义奖品列表和权重
4. 点击「保存」后转盘自动更新

## 定制指南

### 修改默认奖品

在 `popup.js` 中修改 `DEFAULT_PRIZES` 数组：

```javascript
const DEFAULT_PRIZES = [
  { name: '特等奖', weight: 1 },
  { name: '一等奖', weight: 5 },
  { name: '二等奖', weight: 20 },
  { name: '谢谢参与', weight: 100 },
];
```

### 调整转盘颜色

在 `popup.css` 中修改 `WHEEL_COLORS` 对应的背景变量。
